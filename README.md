# Consulta CEP 

Para consultar localidades completas, através do CEP informado

## 🚀 Começando

Digite o seu cep no campo CEP e clique sobre o botão "pesquisar"
Logo em seguida, aparecerá as informações : Nome da Rua, Bairro, Localidade e Uf
